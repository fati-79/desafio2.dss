Sistema web desarrollado en PHP y MySQL que permite registrar usuarios, iniciar sesión y visualizar productos desde una base de datos.

Funciones
Registro de usuarios
Inicio de sesión
Visualización de productos

Instalación

Copiar los archivos en:
C:\xampp\htdocs\techstore
Crear base de datos:
techstore
Importar el archivo:
techstore.sql
Ejecutar en el navegador:
http://localhost/techstore
